# Codex Terminalbefehle Nachschlagewerk

Dieses Dokument fasst die in den Codex-Sitzungen zu diesem Projekt nachweisbar verwendeten Terminalbefehle zusammen. Wiederholungen und dateispezifische Varianten wurden zu einer sinnvollen Grundform zusammengefasst, damit das Dokument als Nachschlagewerk nutzbar bleibt.

Hinweise:
- Gruppen sind alphabetisch sortiert.
- Befehle innerhalb der Gruppen sind alphabetisch sortiert.
- Beispiele orientieren sich an real verwendeten Aufrufen in diesem Projekt.
- In den ausgewerteten Session-Logs wurde kein expliziter VSCode-CLI-Aufruf wie `code ...` gefunden. VSCode war hier die Host-Umgebung, nicht selbst per Terminalbefehl gesteuert.

## Ergänzende Beobachtungen

- In diesem Projekt wurden viele Befehle in fokussierten Kombinationen genutzt, zum Beispiel `git diff ...`, `sed -n ...` und `pio run -e ...`.
- Für ESP32/PlatformIO waren besonders wichtig: `pio run`, `pio run -t upload`, `pio device monitor`, `xtensa-esp32s3-elf-addr2line` und `esptool.py merge_bin`.
- Für Git wurden bevorzugt sichere Varianten genutzt, z. B. `git merge --ff-only`, `git branch -d` und gezieltes `git add` statt pauschalem Stage-All.
- Für Python standen virtuelle Environments, editable Installs, `pytest`-Aufrufe und die `freeze_setup...`-Buildpfade im Vordergrund.
